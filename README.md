# Lodewatch

A free, fast dashboard tracking the world's battery-mineral mines — lithium, cobalt, nickel, and graphite — with production data, ownership, country-risk indicators, and live news.

Live: [lodewatch.com](https://lodewatch.com)

## Stack
- Next.js 16 (App Router)
- React + Tailwind CSS
- Supabase (Postgres)
- MapLibre GL JS (coming soon)
- Claude API for news classification (coming soon)
- Deployed on Vercel

## Status
v0.1 — May 2026. Hardcoded sample data, 5 mines, dashboard skeleton.
